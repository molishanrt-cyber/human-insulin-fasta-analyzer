# Human Insulin FASTA Analyzer

## Objective
To analyze the human insulin gene sequence obtained from NCBI using Python and Biopython.

## Features
- FASTA file parsing
- Sequence metadata extraction
- Sequence length determination
- First and last 50 base visualization
- A/T/G/C nucleotide counts
- GC content calculation
- AT content calculation
- Biological interpretation

## Technologies Used
- Python
- Biopython
- NCBI RefSeqGene Database
